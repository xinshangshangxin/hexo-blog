github project use travis npm publish

  1. 1. 获取 npm token
  2. 2. 配置 travis
  3. 3. 配置项目代码
  4. 4. 提交你的代码到 github
  5. 5. 去 travis 查看是否 开始构建, 并查看构建是否成功
  • 参考文档

    获取 npm token

    1. 在 https://www.npmjs.com/ 注册/登录
    2. 右上角点击头像 => Tokens 进入 Tokens 编辑页
    3. 新增一个token, 记得保存这个token, 后面无法再次查看, 只能新增

    配置 travis

    1. 进入 https://travis-ci.org/, 注册 或者 使用 GitHub登录(右上角Sign in with GitHub)
    2. 进入 https://travis-ci.org/profile/ 页面, 选择 你要自动构建的项目, 左侧可以切换组织项目
    3. 如下图, 勾选开启构建
    4. 点击要构建的项目, 右上角点击 Settings
    5. 在 Environment Variables 中 添加 NPM_EMAIL (你npm账户的邮箱) 和 NPM_TOKEN (上面获取的 token)

    配置项目代码

    在项目中添加 .travis.yml

    提交你的代码到 github

    1. 先commit 你的正常修改
    2. npm version patch/minor/major 更新你的版本
      比如 使用 npm version patch, 会执行
      a) package.json 中插入（更新）的版本号
      b) 创建一个新的提交
      c) 创建一个 git 标签
    3. git push origin master --tag, push你的代码到github上, 同时把tag也

    去 travis 查看是否 开始构建, 并查看构建是否成功

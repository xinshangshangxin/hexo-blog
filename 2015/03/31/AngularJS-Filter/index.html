内置filter

  • number

    以数字格式输出，第二个为可选参数，表示小数点后保留的位数。 包含非数字字符时使用使用可选参数会出现Syntax Error，不使用则输出空字符。

  • currency

    以货币格式输出，包括非数字字符时出现Syntax Error。

  • lowercase/uppercase

    将字符转小写/大写

  • json

    将对象转为json字符串

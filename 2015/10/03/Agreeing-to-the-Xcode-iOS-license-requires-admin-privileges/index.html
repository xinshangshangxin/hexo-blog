Agreeing to the Xcode/iOS license requires admin privileges

2015年10月03日

解决升级了xcode后出现的问题

  1. 打开终端，输入 sudo xcodebuild -license
  2. 终端提示敲回车键(enter)打开许可协议
  3. 终端提示 按下 space 键阅读许可协议，按q 不阅读
  4. 最终，终端会出现三个选项，agree 、print、cancel，输入agree，然后回车

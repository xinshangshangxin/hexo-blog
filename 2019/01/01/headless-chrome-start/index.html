headless 简介

  • Headless browser
    Headless browser 是没有图形界面的浏览器

  • Headless Chrome/Headless Chromium
    允许 Chrome/Chromium 在 headless/server 的环境中使用

  • Headless Firefox
    允许 Firefox 在 headless/server 的环境中使用

  • GoogleChrome/puppeteer
    Google 开源的 headless nodejs API

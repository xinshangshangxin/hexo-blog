    • 问题:

    Error: Java heap space

    解决方案:

    • 编辑文件 /usr/local/Cellar/hadoop/2.7.3/libexec/etc/hadoop/mapred-site.xml 添加 Xmx4096m

    • 或者在运行hadoop之前 export HADOOP_OPTS="-Xmx4096m"

    • 或者对Hadoop 2+, 修改 mapreduce.map.java.opts
